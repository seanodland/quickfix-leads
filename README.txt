=== QUICKFIX QR CODE PACKAGE ===

FILES (all in this folder):
✓ index.html - The lead capture form
✓ style.css  - Mobile-optimized styling
✓ script.js  - Form logic & notifications
✓ qr-main.png - Main QR code (high-res)
✓ qr-analytics.png - QR with tracking params
✓ qr-business-card.png - 144x144 optimized
✓ qr-print.svg - Vector version for print

DEPLOYMENT STEPS:
1. Upload files to your web host
2. Update script.js with your email address
3. Test the form works from your domain
4. Replace 'yourdomain.com/leadform' in QR code links
5. Print QR codes on cards/flyers/signs

USE CASES:
qr-main.png - Yard signs, flyers, large print
qr-business-card.png - Business cards
qr-print.svg - Professional printing, scales to any size

TRACKING:
The analytics QR adds UTM parameters for Google Analytics tracking.
Scan events will appear in your analytics dashboard.
